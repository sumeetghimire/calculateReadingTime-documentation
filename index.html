calculateReadingTime - Documentation

calculateReadingTime is a simple JavaScript package that calculates the estimated reading time of a block of text, dynamically adjusting based on font size.

How to Use

1. Add the CDN

To use the package, add the following CDN script to your HTML file:

<script src="https://cdn.jsdelivr.net/gh/sumeetghimire/calculateReadingTime/index.js"></script>

2. Add Content

Create a content block for which you want to calculate the reading time. You can add any text inside this block:

<div id="content">
  <h1>Sample Text</h1>
  <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit.</p>
  <p>Proin efficitur libero at dictum feugiat.</p>
</div>

3. Add Element to Display Reading Time

Choose an element where you want the calculated reading time to be displayed. For example, you can add a <div> element:

<div id="reading-time"></div>

4. Call the Function

Once the content is loaded, use the following script to call the displayReadingTime function to display the reading time in your target element:

<script>
  window.onload = function() {
    displayReadingTime('reading-time');
  }
</script>

Example

Complete Example Code:

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Reading Time Test</title>
    <style>
      body {
        font-family: Arial, sans-serif;
      }
      #content {
        font-size: 16px;
        line-height: 1.5;
        margin: 20px;
      }
      p {
        font-size: 14px;
      }
    </style>
  </head>
  <body>
    <div id="content">
      <h1>Sample Text</h1>
      <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit.</p>
      <p>Proin efficitur libero at dictum feugiat.</p>
    </div>

    <div id="reading-time"></div>

    <script src="https://cdn.jsdelivr.net/gh/sumeetghimire/calculateReadingTime/index.js"></script>
    <script>
      window.onload = function() {
        displayReadingTime('reading-time');
      }
    </script>
  </body>
</html>

Note: Make sure to adjust the target element for displaying the reading time based on the element you choose in your HTML. You can use any element like a <div>, <span>, or even a <p> element.
